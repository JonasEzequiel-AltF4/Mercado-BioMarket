Projeto de um sistema simples de Mercado, ainda no começo, com listagem e gerenciamento de produtos utilizando HTML, CSS, JavaScript, Node.js e SQLite.
O objetivo é criar um pequeno e-commerce funcional com páginas de produtos, carrinho, login e painel administrativo.

Tecnologias utilizadas

HTML e CSS para a interface
JavaScript para as interações no front-end
Node.js com Express para o backend
SQLite (better-sqlite3) como banco de dados local

Funcionalidades

Cadastro e exclusão de produtos
Listagem de produtos
Sistema de ofertas
Carrinho de compras
Página de login/cadastro e carrinho pendentes ou em simulação.
Integração com banco de dados SQLite
